# 视频轨迹追踪模块-Tracker

这是一个用于在 Gazebo 俯视视频中追踪小车行驶轨迹的 Python 工具，可以从视频中识别黄色小车并绘制带历史轨迹的新视频。

## 依赖库

用 uv 装好依赖

```bash
uv sync
```

## 用法

确保在工作目录下存在 `input/` 文件夹，并将需要处理的视频放于其中。然后运行.

```bash
uv run tracker.py
```

参数说明：
- `--select-input` : 指定待处理视频序号（整数）。
- `--x-min` : 识别区域左边界的比例（0.0 - 1.0），默认 0。
- `--x-max` : 识别区域右边界的比例（0.0 - 1.0），默认 1。
- `--y-min` : 识别区域上边界的比例（0.0 - 1.0），默认 0。
- `--y-max` : 识别区域下边界的比例（0.0 - 1.0），默认 1。

```bash
uv run tracker.py --select-input 1 --x-min 0.1 --x-max 0.9
```
